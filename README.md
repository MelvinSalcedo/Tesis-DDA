**CONTROLES PARA PODER MANIPULAR EL AVATAR.**

**W A S D** -> movimiento del personaje.

**I J K L** -> movimiento de la cámara.

**H** -> ataque débil.

**U** -> ataque fuerte.

**O** -> enfocar al contrincante.

**SPACE** -> rodar o esquivar.

**SHIFT + W A S D** -> correr.

**F** -> activar escudo.
